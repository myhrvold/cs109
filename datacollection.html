Overview

We collected movie scripts from SimplyScripts.com and DailyScript.com. Altogether, we were able to get a total of 396 scripts, as well as their associated IMDB ID numbers from these sites. Next, we used the RottenTomatoes.com API to grab metadata about each movie. We obtained some data that was unavailable on Rotten Tomatoes from IMDB (Used with permission).

Scraping data

We used the BeautifulSoup module in Python to assist with web scraping. For this process, we iterated through each page containing scripts on SimplyScripts.com and DailyScript.com. If a link linked to a script in txt or html format and the website showed valid IMDB number associated with the script, then our program downloads that script and adds the IMDB number to a list. If the script is in HTML format, then we convert it to txt format using the html2text.py file available at www.aaronsw.com/2002/html2text/. Finally, before moving on to the next step, we delete duplicates from the list of IMDB numbers, so that for the next steps we only grab information once for each movie.
